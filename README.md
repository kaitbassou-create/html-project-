#  Luxury Cars – Sport & Luxury Experience

Luxury Cars est un site web statique développé dans le cadre d’un **projet universitaire**.  
Il présente une sélection de **voitures sportives et luxueuses** avec un design élégant basé sur les couleurs **noir, or et argent**.

##  Objectif du projet
- Appliquer les bases du **HTML5**, **CSS3** et **JavaScript**
- Créer un site web **responsive** et moderne
- Organiser un projet web avec plusieurs pages liées entre elles
- Mettre en valeur le thème du **luxe automobile**

---

##  Technologies utilisées
- **HTML5** – Structure des pages
- **CSS3** – Design, mise en page et animations
- **JavaScript** – Interactions simples
- **Google Fonts (Poppins)** – Typographie élégante

---

##  Pages du site
- `main.html` → Page d’accueil
- `maybach.html` → Page Maybach
- `rolls.html` → Page Rolls-Royce
- `911.html` → Page Porsche 911
- `maclaren.html` → Page McLaren
- `MERCEDES.html` → Page Mercedes SLS AMG
- `contact.html` → Page de contact

---

##  Design
- Thème : **Sport & Luxe**
- Couleurs principales :
  - Noir (#000 / #0b0b0b)
  - Or (#d4af37)
  - Argent (#c0c0c0)
- Design responsive (ordinateur, tablette, mobile)
- Galeries d’images pour chaque voiture
